# AnimeMaster 后续版本路线图

## v2.4.0 — 播放体验增强（用户感知度 ★★★★★ 预计 5-7 天）

### 横屏 / 平板适配
- 剧集播放页改为左右分栏：左侧视频 + 右侧剧集列表/简介，充分利用 tablet 和横屏空间
- 视频播放器全屏时支持手势侧滑返回（目前只能点返回按钮）

### 播放器体验打磨
- 长按倍速提示增加视觉反馈（当前只有功能生效，无动画提示）
- 播放器底部控制栏自动隐藏时机可配置（3s / 5s / 永不）
- 播放失败时显示明确的错误原因（"视频源失效" vs "网络超时" vs "格式不支持"）

### 弹幕增强
- 弹幕密度滑块（稀疏 / 正常 / 密集）
- 弹幕透明度调节
- 弹幕显示区域选择（上半屏 / 全屏 / 下半屏）

## v2.5.0 — 在线源智能调度（用户感知度 ★★★★☆ 预计 5-7 天）

### 在线源热更新
- `assets/online_sources.json` 支持从服务器拉取最新版本
- 站点失效自动降级，无需发版即可增删站点
- 在设置页显示"在线源版本"和"上次更新时间"

### 源质量评分体系
- 根据历史成功率、响应速度、清晰度对源进行自动评分
- 搜索结果按评分排序，优先展示高质量源
- 多次失败的源自动降低优先级

### 播放源手动切换
- 当前在线源搜索结果展示每个源的标签（解析速度、清晰度标识）
- 支持手动切换播放源而无需重新搜索

## v2.6.0 — 国际化与可访问性（用户感知度 ★★★★☆ 预计 5-7 天）

### 中英双语
- 迁移到 `flutter_localizations` + ARB 文件
- 设置页增加语言切换（中文 / English）
- AppStrings 所有字符串提供英文翻译
- Bangumi API 响应中的日文/中文标题根据语言偏好优先展示

### 可访问性
- 关键交互元素添加 Semantics 标签（播放按钮、剧集卡片、导航栏）
- 确保所有文字与背景对比度满足 WCAG AA 标准
- 支持系统字体缩放（目前部分布局固定字号）

### 暗色模式完善
- 启动屏适配暗色模式（目前硬编码亮色）
- 详情页头图在暗色模式下降噪处理

## v2.7.0 — 深度链接与导航升级（用户感知度 ★★★☆☆ 预计 5-7 天）

### GoRouter 深度链接
- 迁移到 GoRouter，支持命名路由和深度链接
- 外部链接 `animemaster://detail/12345` 直接打开番剧详情页
- Bangumi 分享链接可跳转到 App 内对应页面
- 浏览器中点击 bgm.tv 链接可选择"在 AnimeMaster 中打开"

### 导航体验
- 详情页返回时保持搜索/列表滚动位置（目前返回后重置）
- 底部导航栏替代目前的顶部工具栏（可选，需设计评审）
- 最近浏览记录（本地存储最近 20 条详情页访问）

## v2.8.0 — 下载体验重构（用户感知度 ★★★★☆ 预计 7-10 天）

### BT 下载引擎优化
- 下载速度图表展示（最近 60 秒速度曲线）
- 支持选择性下载（只下载特定集数，不下载整个种子）
- 下载队列管理：并发数、优先级、WiFi-only 策略
- 做种完成后自动停止（可选配置做种时间/分享率）

### 下载中心 UI
- 下载列表支持长按多选（批量暂停/删除/重试）
- 下载完成通知（系统通知栏显示）
- 存储空间预估（下载前显示所需空间 + 当前剩余空间）

### 边下边播
- 当前仅在下载到足够数据后才播放，优化为更激进的边下边播策略
- 播放时优先下载当前观看位置附近的数据块

## v2.9.0 — 社区功能增强（用户感知度 ★★★☆☆ 预计 5-7 天）

### Bangumi 交互
- 支持在 App 内发表/回复吐槽和评论（目前只读）
- 番剧评分快捷入口（播放完成后弹窗邀请评分）
- 收藏状态快捷切换 widget（详情页一键标记看过/想看）

### 弹幕社区
- 弹幕发送入口（对接 DanDanPlay 投稿 API）
- 本地弹幕缓存管理（清除旧弹幕、查看缓存大小）

## v3.0.0 — 跨平台正式版（用户感知度 ★★★★★ 预计 10-14 天）

### iOS 正式适配
- iOS 播放器手势适配（目前仅 Android 完整）
- iOS 画中画
- App Store 合规审核准备（隐私清单、加密申报）

### Windows / macOS 桌面版
- 窗口尺寸记忆和最小尺寸限制
- 桌面端快捷键（空格暂停、左右方向键快进、F 全屏等）
- 桌面端拖拽打开本地视频文件

### 数据同步
- Bangumi 以外的数据（设置、RSS 源、下载任务）支持 WebDAV 备份/恢复
- 多设备观看进度通过 Bangumi 云同步（利用现有的 collection API）
